# yesCode

AI-assisted software, automation, and product builds.

Repository: <https://github.com/Oyewolesyl/yesCode.git>

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Project images and videos live in `src/assets`. The profiled projects are A Home Realty, YPOD, YPOD Store, DealRadar NG, and NaturePacks.
